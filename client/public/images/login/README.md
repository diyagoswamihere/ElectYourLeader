# Login illustrations

Place your cartoon voting illustrations in this folder so the login page can display them.

Expected filenames (you can replace them with any images, just keep the names):

- `vote-today.png`
- `polling-station.png`
- `polling-place.png`
- `voting-steps.png`
- `voice-matters.png`

If an image is missing, the login page will simply hide that illustration.
